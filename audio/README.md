## 音频规范

### 格式要求
- **格式**：MP3
- **采样率**：44100Hz
- **比特率**：128kbps（BGM可用192kbps）
- **声道**：立体声

### 音量规范
- **BGM**：音量较低（代码中默认0.3），不压住音效
- **音效**：音量适中（代码中默认0.6），清晰突出

### 风格要求
1. **整体风格**：中式清茶风、轻柔舒缓、偏茶馆国风
2. **不吵闹、不刺耳**：适配麻将对局氛围
3. **音效层次分明**：BGM是背景，操作音效要清晰

## 使用方法

1. 将音频文件放入本目录
2. 确保文件名与上表一致
3. 游戏会自动加载并播放

## 代码接口

```javascript
// 播放音效
AudioMgr.play('draw');      // 摸牌
AudioMgr.play('discard');   // 出牌
AudioMgr.play('peng');      // 碰牌
AudioMgr.play('gang');      // 杠牌
AudioMgr.play('huZimo');    // 自摸胡
AudioMgr.play('huDianpao'); // 点炮胡
AudioMgr.play('shou10');    // 凑十达成

// 控制BGM
AudioMgr.playBgm();  // 播放BGM
AudioMgr.stopBgm();  // 停止BGM

// 静音切换
AudioMgr.toggle();   // 返回当前是否开启音效

// 设置音量
AudioMgr.setVolume(0.3, 0.6);  // (BGM音量, 音效音量)
```
